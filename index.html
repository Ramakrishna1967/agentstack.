01 · @observe decorator

A single decorator that instruments any Python function — sync or async, nested or flat. It captures inputs, outputs, token counts, latency, and parent-child span relationships automatically. Zero changes to your agent logic required.

02 · Oxly Engine (FastAPI)

A high-performance single-process FastAPI service. It validates incoming spans, applies rate limits, inspects for security anomalies in real-time, computes per-model costs, and durably stores everything in a local SQLite database.

03 · Dashboard (React + TypeScript)

A Vite-powered React 19 SPA that gives you a live trace explorer, time-series cost charts, security alert feed, and the Time Machine — a step-by-step replay of any past agent run with full span data at every node.

§ 02 · Features
⎈
Real-time Tracing
zero manual work beyond @observe

Capture every LLM call, tool invocation, and function execution as a structured span. Supports sync, async, and deeply nested execution chains.

⟲
Time Machine Replay
step-by-step replay

Step through any past agent execution span-by-span. See what each tool returned and which decision path was taken — without reproducing the failure.

🛡
Security Engine
real-time detection

Detect prompt injection, PII leakage, jailbreak patterns, token explosions, and anomalous behavior in real time. Alerts surface in the dashboard within seconds.

⦸
Auto PII Sanitization
zero config required

SSNs, credit cards, emails, phone numbers, AWS keys, OpenAI keys — detected and redacted from every span before storage automatically.

＄
Cost Analytics
per-model · timeseries

Per-model token counting and USD cost calculation. Track spend across GPT-4, Claude, Gemini — broken down by hour, day, or week.

⚡
Offline Resilience
zero data loss

If the Oxly Engine is unreachable, spans are written to a local SQLite buffer and replayed automatically when connectivity is restored.

§ 05 · Architecture
L1
Your Application — @observe decorator
HTTPS · gzip · exponential backoff retry

sdk-python

Python 3.10+, Pydantic v2. Provides the @observe decorator, local SQLite buffer for offline resilience, automatic PII scrubbing, span batching, and gzip compression.

L2
API Server (FastAPI)
REST + WebSocket + JWT auth

api

FastAPI, SQLite (user store). Exposes REST endpoints for trace queries and a WebSocket channel for live span streaming to the dashboard. JWT tokens with 24h expiry; pbkdf2_sha256 password hashing; brute-force lockout after 5 failures.

L3
Dashboard (React 19 + TypeScript)
Traces · Analytics · Security · Time Machine

dashboard

React 19, TypeScript, Vite. Live trace tree explorer, time-series cost charts, real-time security alert feed, and the Time Machine UI — which lets you scrub backward through any past agent run span-by-span with full input/output data at every step.

§ 06 · Security

Production hardened.
Not an afterthought.

🔒 Non-root containers

All Docker images run as a non-root appuser — no root privileges at runtime.

🔑 JWT Auth

24h token expiry with pbkdf2_sha256 password hashing. Brute-force lockout after 5 failures per 15 minutes.

🛡 API Key Security

SHA-256 hash cache on the hot path · pbkdf2 verification on first use. Keys never stored in plaintext.

🚫 Brute Force Protection

5 failed login attempts per 15-minute window triggers automatic account lockout.

⚡ Rate Limiting

100 requests per minute per IP enforced in-process.

🌐 CORS Allowlist

Explicit origin allowlist. Credentials disabled on the API endpoint. No wildcard origins.

📦 Payload Limits

5MB hard limit enforced on actual request body — not just headers — at the HTTP layer.

🔍 CI Scanning

Bandit static analysis runs on every pull request. Security issues block merge automatically.

🧹 PII Scrubbing

SSNs, card numbers, emails, phone numbers, AWS & OpenAI keys stripped from every span before storage.
